Linked List: 
        In computer science, a linked list is a linear collection of 
        data elements whose order is not given by their physical placement in memory. 
        Instead, each element points to the next. It is a data structure consisting of a 
        collection of nodes which together represent a sequence. Wikipedia





Array: 
        In computer science, an array is a data structure consisting of a 
        collection of elements, of same memory size, each identified by at 
        least one array index or key. An array is stored such that the position 
        of each element can be computed from its index tuple by a mathematical 
        formula. Wikipedia
